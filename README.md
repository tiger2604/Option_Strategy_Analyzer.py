# Option Strategy Analyzer 📈

A Python-based tool to analyze option trading strategies and visualize payoff diagrams.

## Features
- Long Call
- Long Put
- Bull Call Spread
- Bear Put Spread
- Long Straddle
- Payoff Table Output (Pandas)
- Payoff Graph Visualization (Matplotlib)

## Tech Stack
- Python
- NumPy
- Pandas
- Matplotlib

## How to Run
1. Clone the repo
2. Install required libraries
3. Run the script

```bash
python Option_Payoff_Calculator_02.ipynb
